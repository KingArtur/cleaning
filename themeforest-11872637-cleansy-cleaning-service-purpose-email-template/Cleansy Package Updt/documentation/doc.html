3 Using Stampready Builder



The most powerful of cleansy's feature is the easy-to-use stampready builder compatibility. You can easily change the color and image with simple clicks in the editor panel. Learn how to use it :

1. Log In/ Sign Up in Stampready.net

You need to have an account in stampready.net before using this easy drag-n-drop builder. Don't worry, it builder is FREE to use and also you can export your edited version later to HTML package and ready to use in all e-mail clients.

2. Upload Zip File & Edit the campaign

Create your campaign by click in 'NEW CAMPAIGN' in the top-right of stampready dashboard, next step is to click 'upload template' (still in the top-right of the dashboard) and then choose 'ZIP FILE'. Uploade the zip file inside of stampready template in the downloaded package. After upload finished and preview is showing in your browser, click 'USE TEMPLATE', last step is give a name of your campaign and start editing.

7 PSD Template Editing

Basic Needed :

- First, you need Adobe Photoshop CS3 or Later Version
- make sure you have 'Open Sans' font. If you do not have it, download it here
- Basic Photoshop skills

How to Edit :


- Open files in Photoshop. Make sure you show your layer like image below.



- Change Color : All content (except images) basically built from vector shaped, you can easily change the color with double click in the layer thumbnails

- Replace Images : All images is based with smart-object, you can easily replace the image with double click in the layer thumbnails

- Change pattern (optional) : Open file called “pattern” in the file package to photoshop. The file is based by vector shape, you can edit it color with double click in layer tumbnails. After you change the pattern color, go to “edit > define pattern”, name it so you can easily detect the file. Then go back to your file, double click “pattern overlay” in your photoshop layer, than change pattern with latest pattern that you define before.
